# CineTrackr

### Course: CI/CD

CineTrackr is a personal movie and TV series tracking application.

The goal of the project is to provide a simple way to keep track of what has been watched, what is currently being watched, and what is planned for later. The application is split into a Spring Boot backend and a React frontend.

#### Project Structure
backend/ 

Spring Boot application.

frontend/ 

React application built with Vite that provides the user interface.

### Technologies

#### Backend
- Java
- Spring Boot
- Maven

#### Frontend

- React
- Vite
- JavaScript

#### Current Status
This project is currently under active development. The structure is in place and core functionality is being implemented.
